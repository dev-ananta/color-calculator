# Color Calculator

Web-based Utility Tool for Developers & Designer to manipulate, convert, and analyze colors in real-time.

## Project Structure
```
mac-google-messages/
│── README.md → Documentation & Information
│── LICENSE.md → License pertaining to Copyright Information revolving this project/repository.
│── index.html → Primary Entry Point & UI Structure
│── calculator.html → Specialized Calculation Module 
│── script.js → Color Conversion & Manipulation Logic
│── styles.css → Component & Layout Styling
```

## Project Features

- **Multi-Format Conversion**: **HEX** & **RGB**.
- **Color Mixing**: Blend 2 Colors using mathematical modes. (Average, Mutliply, Screen, Overlay, etc.)
- **Accessibility Tools**: Integrate **WCAG 2.1 Contrast Checker** in order to ensure your color(s) meet accessibility standards.
- **Palette Generation**: Automatically generate **Complemenatary**, **Analogous**, an d**Triadic** color palettes.
- **Live Preview**: Visual representation of colors change within an instant via CSS code generation.
